\# 🚀 Employee Management System



A simple Employee Management application built with Flask and SQLite.



\## Features



\- Add Employee

\- View Employees

\- Search Employees

\- Update Employee

\- Delete Employee



\## Tech Stack



\- Python

\- Flask

\- SQLite

\- HTML



\## Run



```bash

python app.py

```



Visit:



```

http://localhost:5000

```
